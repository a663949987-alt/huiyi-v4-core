# 会意 v4.1.3-lan 局域网更新修复说明

## 结论

本次更新已完成，版本为 `4.1.3-lan`，版本号 `413`。

最新 APK：

`outputs/huiyi-v4.1.3-lan-debug.apk`

局域网更新包：

`outputs/update_server/huiyi-v4.1.3-lan-debug.apk`

局域网更新清单：

`outputs/update_server/latest.json`

当前电脑局域网地址：

`http://192.168.31.243:8787/latest.json`

## 修复内容

1. 修复电脑端局域网更新服务打开但不返回内容的问题。
2. 修复 `latest.json` 可能带 BOM，导致手机端 JSON 解析失败的问题。
3. App 允许局域网 `http://` 下载更新包。
4. 版本号从 `4.1.2-lan (412)` 升到 `4.1.3-lan (413)`，手机会识别为新版本。
5. 已重新生成 APK 和 `latest.json`。
6. 已提交并推送 GitHub：`80b9f67 Fix LAN update delivery`。

## 需要实机测试

需要。这个功能涉及手机 Wi-Fi、电脑局域网服务、Android 下载、APK 安装器和系统权限，本地单元测试不能完全覆盖。

## 实机测试步骤

1. 先安装 `huiyi-v4.1.3-lan-debug.apk`。
2. 手机浏览器打开：

   `http://192.168.31.243:8787/latest.json`

   能看到 `4.1.3-lan` 的 JSON，说明手机能连到电脑更新服务。
3. 打开会意 App，进入设置页。
4. 更新地址可以留空，点“检查”。
5. 如果自动发现失败，就手动填：

   `http://192.168.31.243:8787/latest.json`

6. 点“检查”，应显示发现版本。
7. 点“下载”，应拉起系统安装确认。
8. 如果系统提示“不允许安装未知来源应用”，给会意开启安装权限后再试一次。

## 预期结果

- 手机能访问电脑的 `latest.json`。
- App 能检查到局域网更新。
- App 能下载 APK。
- 下载后能进入系统安装确认页。

## 如果仍失败

优先看手机浏览器能不能打开：

`http://192.168.31.243:8787/latest.json`

如果浏览器也打不开，说明是电脑防火墙或路由器隔离挡住了手机访问电脑的 `8787` 端口。
